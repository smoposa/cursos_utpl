Cursos Disponibles

Administración de Windows Server

La estructura del curso está basada en un 90% en el temario de la certificación “MCSA: Windows Server”, tomando como referencia sus guías oficiales de estudio y además varios textos nuevos de administración de Windows Server. Este material se enriquece además con 14 años de experiencia del instructor en la administración de redes Microsoft y 5 años impartiendo el curso de Windows Server.


Administración de Microsoft Azure

Este curso enseña a los profesionales de TI cómo administrar sus suscripciones de Azure, proteger identidades, administrar la infraestructura, configurar redes virtuales, conectar Azure y sitios locales, administrar el tráfico de red, implementar soluciones de almacenamiento, crear y escalar máquinas virtuales, implementar aplicaciones web y contenedores, haga copias de seguridad y comparta datos, y monitoree su solución.


Bases de Datos desde cero con SQL

Ayudar al estudiante a conocer los elementos que componen el IDE de administración SQL server, sintaxis de comandos utilizados para realizar la creación de una base de datos, consultas, vistas, procedimientos almacenados, ingreso, eliminación y actualización de registros.
